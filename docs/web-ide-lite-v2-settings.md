# Web IDE Lite v2 - 设置功能文档

## 概述

在 Web IDE Lite v2 的左下角状态栏添加了设置按钮（⚙️），点击可打开设置面板，支持配置编辑器的各种常见设置。

## 功能特性

### 1. 设置分类

设置面板分为 5 个分类：

| 分类 | 图标 | 说明 | 设置项数量 |
|------|------|------|-----------|
| 编辑器 | 📝 | 编辑器相关设置 | 10 项 |
| 外观 | 🎨 | 界面外观设置 | 4 项 |
| 文件 | 📁 | 文件管理设置 | 5 项 |
| 快捷键 | ⌨️ | 快捷键配置 | 7 项 |
| 其他 | ⚙️ | 其他杂项设置 | 6 项 |

### 2. 编辑器设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 字体大小 | 数字 | 14px | 编辑器字体大小（8-72px） |
| 行高 | 数字 | 1.5 | 文本行高（1-3） |
| 自动换行 | 布尔 | false | 是否自动换行 |
| Tab 大小 | 数字 | 2 | Tab 相当于几个空格（1-8） |
| 使用空格代替 Tab | 布尔 | true | 按 Tab 时插入空格 |
| 显示行号 | 布尔 | true | 是否显示行号 |
| 高亮当前行 | 布尔 | true | 高亮显示光标所在行 |
| 括号匹配 | 布尔 | true | 自动匹配括号 |
| 自动闭合括号 | 布尔 | true | 输入左括号自动添加右括号 |
| 自动缩进 | 布尔 | true | 自动缩进 |

### 3. 外观设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 主题 | 选择 | dark | 深色/浅色/午夜蓝/Monokai |
| 侧边栏宽度 | 数字 | 288px | 侧边栏宽度（200-500px） |
| 显示状态栏 | 布尔 | true | 是否显示底部状态栏 |
| UI 字体大小 | 数字 | 14px | 界面字体大小（12-24px） |

### 4. 文件设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 自动保存 | 布尔 | true | 是否启用自动保存 |
| 自动保存延迟 | 数字 | 2000ms | 内容变化后延迟保存（500-10000ms） |
| 定时保存间隔 | 数字 | 30000ms | 定时保存间隔（10-300 秒） |
| 最大备份数 | 数字 | 10 | 最多保存的备份数量（1-50） |
| 默认语言 | 选择 | plaintext | 新建文件的默认语言 |

### 5. 快捷键设置

| 快捷键 | 默认值 | 说明 |
|--------|--------|------|
| 保存 | Ctrl+S | 保存当前文件 |
| 新建文件 | Ctrl+N | 创建新文件 |
| 打开文件 | Ctrl+O | 打开文件 |
| 搜索 | Ctrl+F | 搜索文本 |
| 替换 | Ctrl+H | 替换文本 |
| 切换侧边栏 | Ctrl+B | 显示/隐藏侧边栏 |
| 切换主题 | Ctrl+Shift+T | 切换深色/浅色主题 |

### 6. 其他设置

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 删除前确认 | 布尔 | true | 删除文件前是否确认 |
| 平滑滚动 | 布尔 | true | 是否启用平滑滚动 |

## 使用方法

### 打开设置面板

1. 点击左下角状态栏的 ⚙️ 按钮
2. 或使用快捷键（待实现）

### 修改设置

1. 在左侧选择设置分类
2. 在右侧修改具体设置值
3. 设置会自动保存

### 设置类型说明

#### 布尔类型
点击开关按钮切换 开/关 状态

#### 数字类型
- 直接输入数值
- 或使用上下箭头调整
- 有最小/最大值限制

#### 选择类型
点击下拉框选择选项

#### 快捷键类型
1. 点击"录制"按钮
2. 按下新的快捷键组合
3. 自动保存

### 批量操作

#### 导出设置
1. 点击设置面板底部的"📤 导出设置"按钮
2. 选择保存位置
3. 保存 `web-ide-settings.json` 文件

#### 导入设置
1. 点击设置面板底部的"📥 导入设置"按钮
2. 选择之前导出的 JSON 文件
3. 设置自动应用

#### 重置设置
1. 点击设置面板右上角的"🔄 重置"按钮
2. 确认重置操作
3. 所有设置恢复默认值

## 技术实现

### 文件结构

```
public/web-ide-lite-v2/
├── index.html                  # 设置面板 UI
├── js/
│   ├── state.js                # 设置状态
│   ├── composables.js          # 设置功能集成
│   └── composables/
│       └── settings.js         # 设置管理模块
```

### 设置存储

设置使用 `localStorage` 存储，键名为 `web-ide-settings`。

```javascript
// 存储格式
{
  "editor": {
    "fontSize": 14,
    "lineHeight": 1.5,
    ...
  },
  "appearance": {
    "theme": "dark",
    ...
  },
  ...
}
```

### API 函数

```javascript
// 初始化设置
initSettings();

// 获取单个设置
getSetting('editor.fontSize');

// 获取所有设置
getAllSettings();

// 设置单个值
setSetting('editor.fontSize', 16);

// 批量设置
setSettings({ editor: { fontSize: 16 } });

// 重置设置
resetSettings();           // 重置所有
resetSettings('editor');   // 重置指定分类

// 导出/导入
exportSettings();          // 导出为 JSON 字符串
importSettings(json);      // 从 JSON 导入

// 监听变更
watchSettings((key, value) => {
  console.log(`设置变更：${key} = ${value}`);
});
```

### 设置应用

当设置变更时，会自动应用：

```javascript
// 主题变更
if (key === 'appearance.theme') {
  setTheme(value);
}

// 侧边栏宽度
if (key === 'appearance.sidebarWidth') {
  sidebar.style.width = `${value}px`;
}

// 状态栏可见性
if (key === 'appearance.statusBarVisible') {
  statusBar.style.display = value ? 'flex' : 'none';
}

// 自动保存重新初始化
if (key === 'files.autoSave') {
  destroyAutoSave();
  initAutoSaveFeature();
}
```

## 注意事项

1. **设置自动保存**：修改设置后立即保存到 localStorage
2. **刷新不丢失**：刷新页面后设置会自动恢复
3. **跨标签同步**：同一域名下的所有标签页共享设置
4. **浏览器限制**：清除浏览器缓存会丢失设置
5. **导出备份**：建议定期导出设置备份

## 故障排查

### 设置不生效

1. 检查浏览器控制台是否有错误
2. 尝试刷新页面
3. 清除 localStorage 后重试

### 无法导出/导入

1. 检查浏览器权限
2. 确保 JSON 文件格式正确
3. 尝试更换浏览器

### 设置丢失

1. 检查是否清除了浏览器数据
2. 从备份文件导入
3. 重置为默认设置
