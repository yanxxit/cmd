# File Viewer 待优化方向（第五轮 - 2026-03-10）

## 概述

本文档列出了本地文件查看器 (file-viewer) 项目在已有四轮优化基础上的 3 个新增待优化方向。

**检查时间**: 2026-03-10  
**实施状态**: ✅ 部分完成（快速预览和刷新按钮已完成）

---

## ✅ 优化方向一：文件预览增强 - 快速查看模式（已完成）

### 问题描述
当前查看文件需要打开模态框，对于快速预览不够便捷：
- 查看文件必须打开模态框
- 无法快速浏览多个文件
- 模态框关闭后重新打开需要重新加载

**用户需求**:
- 空格键快速预览（类似 macOS Quick Look）
- 预览窗口半透明悬浮
- 支持快速切换上一个/下一个文件

### 优化目标
- 空格键打开/关闭快速预览
- 预览窗口半透明悬浮显示
- 支持方向键切换文件
- 点击外部关闭预览

### 实现内容
- ✅ 快速预览窗口 HTML 和 CSS
- ✅ 空格键打开预览
- ✅ 左右方向键切换文件
- ✅ 点击遮罩关闭预览
- ✅ 支持文本和图片预览

### 优先级
🔴 高（提升浏览效率）

---

## ⏸️ 优化方向二：常用目录收藏功能（待实现）

### 问题描述
当前访问深层目录需要多次点击，无法快速定位：
- 无法收藏常用目录
- 每次访问深层目录需要逐层进入
- 没有历史记录功能

**用户需求**:
- 收藏常用目录
- 快速访问收藏目录
- 查看访问历史

### 优化目标
- 右键菜单支持收藏目录
- 侧边栏显示收藏目录列表
- 支持添加/删除收藏
- localStorage 持久化存储

### 优先级
🟡 中（提升用户体验）

---

## ✅ 优化方向三：文件预览 - 刷新按钮（已完成）

### 问题描述
当前查看文件时，如果文件被外部修改，无法刷新查看最新内容：
- 文件被外部修改后无法刷新
- 需要关闭模态框重新打开
- 无法自动检测文件变化

**用户需求**:
- 模态框内刷新按钮
- 快捷键刷新（F5）
- 显示最后刷新时间

### 优化目标
- 模态框添加刷新按钮
- F5 快捷键刷新文件内容
- 刷新后显示 Toast 提示

### 实现内容
- ✅ 模态框刷新按钮
- ✅ F5 快捷键支持
- ✅ 刷新函数 `refreshCurrentFile()`
- ✅ Toast 成功/失败提示

### 优先级
🟢 低（功能增强型优化）

## 使用说明

### 快速预览
| 操作 | 功能 |
|------|------|
| `空格键` | 打开/关闭快速预览 |
| `←` / `→` | 上一个/下一个文件 |
| `Esc` | 关闭预览 |
| 点击遮罩 | 关闭预览 |

### 刷新文件
| 操作 | 功能 |
|------|------|
| `F5` | 刷新当前文件 |
| 点击🔄按钮 | 刷新当前文件 |
