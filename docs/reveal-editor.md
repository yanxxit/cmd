# Reveal.js 演示文稿编辑器

基于 [reveal.js](https://revealjs.com/) 和 Markdown 的在线演示文稿编辑器，让你轻松创建精美的幻灯片。

## 访问地址

启动服务器后访问：http://127.0.0.1:3000/reveal-editor/

## 功能特点

- 📝 **Markdown 编辑** - 使用熟悉的 Markdown 语法编写内容
- 🎨 **多主题支持** - 12 种精美主题随意切换
- 👁️ **实时预览** - 编辑内容即时渲染
- 📥 **导出 HTML** - 生成独立的 HTML 文件，随处播放
- 🎯 **快速插入** - 一键插入幻灯片、代码、表格等
- ⌨️ **快捷键** - 支持撤销/重做、格式化等操作
- 📋 **预设模板** - 基础、技术分享、产品介绍、教学课件

## 快速开始

### 1. 选择模板

点击顶部工具栏的模板按钮：
- 📋 **基础模板** - 简单的演示文稿结构
- 💻 **技术分享** - 适合技术演讲
- 🚀 **产品介绍** - 产品展示模板
- 📚 **教学课件** - 教育培训模板

### 2. 编辑内容

在左侧编辑器中使用 Markdown 语法编写：

```markdown
---

## 幻灯片标题

内容...

---

## 下一张幻灯片

更多内容...
```

### 3. 切换主题

从主题选择器中选择喜欢的主题：
- ⚫ 黑色主题 (black)
- ⚪ 白色主题 (white)
- 🏆 联盟主题 (league)
- ☁️ 天空主题 (sky)
- 🌙 月光主题 (moon)
- 🎨 太阳能主题 (solarized)
- 🧛 吸血鬼主题 (dracula)
- 等等...

### 4. 播放演示

点击 **▶️ 播放** 按钮，在新窗口中全屏播放演示文稿。

### 5. 导出 HTML

点击 **📥 导出 HTML** 按钮，下载独立的 HTML 文件。

## Markdown 语法

### 基础语法

#### 幻灯片分隔

使用 `---` 分隔不同的幻灯片：

```markdown
---

## 第一张幻灯片

---

## 第二张幻灯片
```

#### 标题

```markdown
## 幻灯片标题

### 副标题

#### 小标题
```

#### 文本格式

```markdown
**粗体文本**

*斜体文本*

~~删除线~~

[链接](https://example.com)
```

#### 列表

```markdown
## 无序列表

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

## 有序列表

1. 第一步
2. 第二步
3. 第三步
```

#### 代码块

````markdown
## 代码示例

```javascript
function hello() {
  console.log("Hello World!");
}
```

```python
print("Hello Python!")
```
````

#### 引用

```markdown
> 这是一段引用文本
> 
> — 作者名
```

#### 表格

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25 | 北京 |
| 李四 | 30 | 上海 |
```

#### 图片

```markdown
![图片描述](图片 URL)
```

### 高级功能

#### 背景颜色

```markdown
<!-- .slide: data-background="#ff0000" -->

## 红色背景的幻灯片
```

#### 背景渐变

```markdown
<!-- .slide: data-background-gradient="linear-gradient(135deg, #667eea 0%, #764ba2 100%)" -->

## 渐变背景
```

#### 碎片动画

```markdown
## 动画效果

<span class="fragment">第一项（逐步显示）</span>

<span class="fragment">第二项</span>

<!-- 或使用 HTML -->
<p class="fragment">第三项</p>
```

#### 背景图片

```markdown
<!-- .slide: data-background-image="image.jpg" -->

## 图片背景
```

#### 背景视频

```markdown
<!-- .slide: data-background-video="video.mp4" -->
```

#### 代码高亮

使用语言标识符启用代码高亮：

````markdown
```javascript
// JavaScript 高亮
const x = 10;
```

```python
# Python 高亮
x = 10
```

```css
/* CSS 高亮 */
body { color: red; }
```
````

| `↓` / `J` | 下一张 |
| `Home` | 第一张 |
| `End` | 最后一张 |
| `N` | 下一张（所有动画） |
| `P` | 上一张 |
| `Space` | 下一张 |
| `Shift+Space` | 上一张 |
| `O` | 概览模式 |
| `T` | 打开新标签页显示当前幻灯片 |

## 导出 HTML 使用

导出的 HTML 文件是独立的，可以：

1. **本地播放** - 直接用浏览器打开
2. **网络分享** - 上传到服务器
3. **离线演示** - 无需网络环境
4. **嵌入网站** - 通过 iframe 嵌入

### 自定义导出

如需自定义导出的 HTML，可以：

1. 导出基础 HTML
2. 修改 `<head>` 部分添加自定义样式
3. 添加额外的插件或脚本
4. 修改 reveal.js 配置

## 技巧与建议

### 内容组织

1. **每张幻灯片一个主题** - 保持内容聚焦
2. **使用标题层级** - 清晰的视觉层次
3. **适度使用动画** - 不要过度分散注意力
4. **代码简洁** - 只展示关键代码

### 视觉设计

1. **选择合适的主题** - 匹配演讲内容
2. **使用背景色强调** - 突出重要内容
3. **保持一致性** - 统一的风格
4. **留白很重要** - 不要堆砌内容

### 演讲技巧

1. **使用演讲者视图** - 按 `S` 键打开
2. **准备备注** - 在 HTML 中添加 notes
3. **练习翻页** - 熟悉快捷键
4. **备份导出文件** - 防止网络问题

## 常见问题

### Q: 如何添加演讲者备注？

A: 在 Markdown 中添加：

```markdown
## 幻灯片内容

<!-- .element: class="notes" -->
这是只有演讲者能看到的备注
```

### Q: 如何创建垂直幻灯片？

A: 使用 `---` 创建水平分隔，在幻灯片内使用 `--` 创建垂直分隔：

```markdown
---

## 主幻灯片 1

--

### 垂直幻灯片 1.1

--

### 垂直幻灯片 1.2

---

## 主幻灯片 2
```

### Q: 如何调整幻灯片布局？

A: 使用 reveal.js 的布局类：

```markdown
<!-- .slide: data-background-color="#000" -->

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 20px;">
  <div>左侧内容</div>
  <div>右侧内容</div>
</div>
```

### Q: 导出的 HTML 太大？

A: 可以：
1. 使用本地 CDN 资源
2. 压缩 HTML 文件
3. 移除不需要的插件
