# JSON 对比工具

## 概述

一个功能完善的 Web 版 JSON 对比工具，支持差异高亮、统计分析、导出等功能。

## 访问地址

```
http://127.0.0.1:3000/json-diff/
```

## 功能特性

### 1. 基础功能
- ✅ **左右对比** - 分屏显示两个 JSON 的差异
- ✅ **自动格式化** - 自动美化 JSON 格式
- ✅ **拼音排序** - 按 Key 的中文拼音自动排序
- ✅ **实时验证** - 输入时自动验证 JSON 格式

### 2. 差异高亮
- 🔴 **红色标记** - 对方不存在的字段（新增/删除）
- 🔵 **蓝色标记** - 值不同的字段（修改）
- 🟢 **绿色标记** - 相同的字段

### 3. 统计分析
- 📊 总节点数统计
- 📊 相同节点数统计
- 📊 修改节点数统计
- 📊 新增/删除节点数统计

### 4. 交互功能
- 🔄 **交换左右** - 一键交换两个 JSON 的位置
- 📖 **展开全部** - 展开所有节点
- 📕 **收起全部** - 收起所有节点
- 🙈 **隐藏相同** - 隐藏相同的节点，只显示差异
- 📋 **复制结果** - 复制差异数据到剪贴板
- 📥 **导出差异** - 导出差异数据为 JSON 文件

## 使用方法

### 步骤 1: 输入 JSON
在左右两个输入框中分别粘贴或输入要对比的 JSON 数据。

> 💡 **提示**：粘贴 JSON 后会自动格式化并按 Key 的字母/拼音排序，无需手动操作。

### 步骤 2: 点击对比
点击「🔍 开始对比」按钮，系统会自动：
1. 验证 JSON 格式
2. 按拼音排序 Key
3. 计算差异
4. 显示统计信息
5. 渲染带高亮的结果

### 步骤 3: 查看差异
- 红色背景 + 🗑️ 图标：该字段在对方中不存在
- 蓝色背景 + ⚠️ 图标：该字段的值不同
- 绿色背景：相同的字段

> 💡 **提示**：切换「隐藏相同部分」开关，可以**真正过滤掉**相同的字段，只显示有差异的数据。这不是简单的 CSS 隐藏，而是从数据层面过滤。

### 步骤 4: 操作结果
- 点击「展开全部」/「收起全部」控制节点展开状态
- 点击「导出差异数据」下载 JSON 文件
- 点击「复制结果」复制到剪贴板

## 技术实现

### 核心算法

1. **拼音排序**
```javascript
Object.keys(obj).sort((a, b) => {
  return a.localeCompare(b, 'zh-Hans-CN', { sensitivity: 'accent' });
});
```

2. **差异计算**
- 递归遍历对象的所有键
- 比较键的存在性和值
- 区分新增、修改、删除三种状态

3. **过滤相同字段**
```javascript
function filterSameFields(data, compareData, side) {
  // 对象：过滤相同的字段
  const result = {};
  for (const key of Object.keys(data)) {
    const value = data[key];
    const compareValue = compareData ? compareData[key] : undefined;
    const hasCompare = compareData && key in compareData;

    if (hasCompare) {
      // 两边都有且值相同 → 过滤掉（不添加到 result）
      // 两边都有但值不同 → 保留
      // 两边都是对象 → 递归过滤
    } else {
      // 只有这边有 → 保留（表示被删除或新增的）
    }
  }
  return result;
}
```

4. **差异高亮**
- 红色：`json-diff-added` - 对方不存在
- 蓝色：`json-diff-modified` - 值不同
- 绿色：`json-diff-same` - 相同

## 导出格式

导出的 JSON 文件包含以下结构：

```json
{
  "generatedAt": "2026-03-20T12:00:00.000Z",
  "stats": {
    "total": 20,
    "same": 10,
    "modified": 5,
    "added": 5
  },
  "diff": {
    "fieldName": {
      "_action": "modified",
      "_leftValue": "原值",
      "_rightValue": "新值"
    }
  }
}
```

## 注意事项

1. **JSON 格式** - 必须输入有效的 JSON 格式
2. **大文件处理** - 对于超大 JSON 文件，渲染可能需要一些时间
3. **数组对比** - 数组按索引位置对比，不按内容对比
4. **类型差异** - 如果类型不同（如对象 vs 数组），会标记为修改
