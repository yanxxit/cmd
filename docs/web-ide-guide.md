# Web IDE 使用文档

> 📅 创建时间：2026-03-13  
> 📋 技术栈：Vue 3 + Monaco Editor + Tailwind CSS

---

## 🎯 功能概述

基于 Monaco Editor（VS Code 使用的编辑器）的 Web 在线代码编辑器，支持：
- ✅ 多种语言语法高亮
- ✅ 智能代码补全
- ✅ 文件拖拽上传
- ✅ 本地文件保存
- ✅ 多标签页编辑
- ✅ 暗色主题
- ✅ 实时光标位置显示

## 🔧 功能说明

### 1. 语法高亮

**自动识别：**
- 根据文件扩展名自动识别语言
- 应用对应的语法高亮规则
- 支持智能代码补全

**示例：**
```javascript
// JavaScript
function greet(name) {
  return `Hello, ${name}!`;
}
```

```python
# Python
def greet(name):
    return f"Hello, {name}!"
```

```go
// Go
func greet(name string) string {
    return fmt.Sprintf("Hello, %s!", name)
}
```

### 2. 文件管理

**左侧文件浏览器：**
- 按语言分类显示文件
- 文件夹可展开/折叠
- 点击文件打开编辑

**支持的操作：**
- 📄 新建文件
- 📂 打开文件
- 💾 保存文件
- × 关闭标签页

### 3. 拖拽上传

**拖拽方式：**
1. 直接拖拽文件到编辑器区域
2. 释放后自动打开
3. 自动分类到对应语言文件夹

**支持：**
- 单文件拖拽
- 多文件拖拽
- 文件夹拖拽（需浏览器支持）

### 4. 多标签编辑

**标签页功能：**
- 同时打开多个文件
- 点击标签切换
- 修改指示器（●）
- 关闭按钮（×）

### 5. 本地保存

**保存方式：**
1. 点击"💾 保存"按钮
2. 快捷键 `Ctrl+S` / `Cmd+S`
3. 文件自动下载到本地

## 🐛 常见问题

### 1. Monaco Editor 加载失败

**原因：** CDN 访问问题

**解决：**
- 检查网络连接
- 刷新页面重试
- 清除浏览器缓存

### 2. 文件保存后找不到

**原因：** 浏览器默认下载位置

**解决：**
- 检查浏览器下载文件夹
- 查看浏览器下载历史
- 设置固定下载位置

### 3. 语法高亮不生效

**原因：** 文件扩展名未识别

**解决：**
- 确保文件名包含正确扩展名
- 如：`script.js`, `app.py`
