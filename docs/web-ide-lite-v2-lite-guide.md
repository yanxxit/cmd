## 🎯 功能概述

Web IDE Lite v2 精简版是一个轻量级的在线代码编辑器，专注于核心编辑功能：

**核心功能：**
- ✅ 打开本地目录/文件
- ✅ 语法高亮（10+ 语言）
- ✅ 代码编辑
- ✅ 多标签页
- ✅ 文件/文件夹管理
- ✅ 右键菜单
- ✅ 主题切换（深色/浅色）
- ✅ 文件搜索
- ✅ 本地保存/下载

**已移除的高级功能：**
- ❌ 设置面板
- ❌ 自动保存
- ❌ 快捷键管理
- ❌ 全局搜索
- ❌ 最近文件
- ❌ 编辑器右键菜单
- ❌ 终端集成

## 🚀 快速开始

### 访问地址

```bash
# 启动服务
x-static

# 访问精简版 IDE
http://127.0.0.1:3000/web-ide-lite-v2/
```

### 基本操作

**打开目录：**
1. 点击 "📂 打开目录" 按钮
2. 选择本地文件夹
3. 自动加载目录结构

**创建文件：**
1. 点击 "📄 新建" 按钮
2. 输入文件名（如 `test.js`）
3. 自动打开编辑

**保存文件：**
- 快捷键：`Ctrl+S` / `Cmd+S`
- 点击 "💾 保存" 按钮

**切换语言：**
- 右下角状态栏选择语言
- 自动应用语法高亮

**切换主题：**
- 点击右上角 🌙/☀️ 按钮

## 🐛 常见问题

### 1. 无法打开目录？

**原因：** 浏览器不支持 File System Access API

**解决：** 使用 Chrome/Edge 等现代浏览器

### 2. 高亮不生效？

**原因：** 语言未加载

**解决：** 切换语言后等待自动加载

### 3. 保存失败？

**原因：** 文件权限问题

**解决：** 使用"下载文件"功能保存到本地
