# 24 Paste Parser Parse Mode

## 背景
`public/paste-parser/index.html` 首页已经具备自动识别多种粘贴内容的能力，但当前解析流程仍是单一路径：只能依赖内容探测结果，无法在粘贴前主动指定“按某一类型强制解析”。这会影响边界场景下的可控性，例如用户希望把一段文本强制按 Markdown、JSON 或代码方式查看。

## 目标
- 默认保持现有自动识别行为，并在界面上明确展示为“自动模式”。
- 支持用户先选择某种类型，再强制按该类型执行解析。
- 将“自动/强制”分发规则抽为可测试纯逻辑，避免事件处理函数继续膨胀。
- 不回退现有表格、富文本、图片、文件、URL、JWT、Base64、颜色等能力。
