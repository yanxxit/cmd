# 20 2048 ESM Refactor

## 背景
`public/2048/index.html` 当前是单文件页面，包含大量内联样式、渲染逻辑、存储逻辑、输入绑定和游戏规则实现，不利于后续维护。

## 目标
- 将 2048 页面拆成 `index.html + css/ + js/` 的原生 ESM 结构。
- 抽离核心游戏规则为可测试模块。
- 保持现有玩法、撤销、统计、键盘与触摸交互不回退。

## 计划
- [ ] 提取并测试核心移动/合并逻辑
- [ ] 拆分存储层、渲染层、输入层和应用装配层
- [ ] 将样式拆分到 `css/` 目录
- [ ] 保留首页访问入口和原有页面路径
- [ ] 完成目标测试与仓库级验证
