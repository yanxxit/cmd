## 🎯 功能概述

Markmap 思维导图是一个基于 Markdown 语法的思维导图工具，支持实时预览和多种导出功能。

**核心功能：**
- ✅ Monaco Editor 代码编辑
- ✅ 实时渲染思维导图
- ✅ 支持 Markdown 语法
- ✅ 支持代码高亮
- ✅ SVG/PNG 格式导出
- ✅ 缩放控制
- ✅ 全屏模式
- ✅ 示例代码快速加载

## 💡 使用技巧

### 创建思维导图

1. **确定主题**
   ```markdown
   # 我的思维导图
   ```

2. **添加分支**
   ```markdown
   ## 主要分支 1
   ## 主要分支 2
   ```

3. **添加子项目**
   ```markdown
   - 子项目 1
     - 详细项 1
     - 详细项 2
   ```

### 最佳实践

**结构清晰：**
```markdown
# 中心主题

## 分支 1
- 子项 1
- 子项 2

## 分支 2
- 子项 3
```

**简洁明了：**
- 每个节点文字不宜过长
- 使用关键词而非长句
- 合理控制层级深度（建议不超过 5 层）

**使用格式：**
- 重要内容使用 **粗体**
- 代码使用 `行内代码`
- 链接直接嵌入

## 🐛 常见问题

### 1. 思维导图不显示

**原因：** Markdown 语法错误

**解决：**
- 确保使用正确的标题语法（# ## ###）
- 检查列表格式（- 或 1.）
- 查看错误提示信息

### 2. 渲染速度慢

**原因：** 内容过多或过深

**解决：**
- 简化思维导图结构
- 减少嵌套层级
- 拆分多个思维导图

### 3. 导出的 PNG 模糊

**原因：** 原始 SVG 尺寸较小

**解决：**
- 使用 SVG 格式（矢量图）
- 或先放大再导出

### 4. 代码块不显示

**原因：** 代码块语法错误

**解决：**
````markdown
正确：
```javascript
code
```

错误：
```
code
```
````
