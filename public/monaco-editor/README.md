## 🎯 功能概述

Monaco Editor IDE 是一个基于 Monaco Editor（VS Code 同款编辑器）的在线代码编辑器，提供完整的文件管理和编辑功能。

**核心功能：**
- ✅ Monaco Editor 完整功能
- ✅ 智能代码补全
- ✅ 多光标编辑
- ✅ 文件/文件夹拖拽排序
- ✅ 标签页拖拽排序
- ✅ 命令面板（Ctrl+Shift+P）
- ✅ 全文搜索
- ✅ 右键菜单
- ✅ 最近文件
- ✅ 打开的编辑器列表
- ✅ 语言切换（20+ 种）
- ✅ 菜单栏

## 📁 文件管理

### 打开目录

**方法一：点击按钮**
1. 点击 "📂 打开目录" 按钮
2. 选择本地文件夹
3. 自动加载目录结构

**方法二：菜单栏**
1. 点击 "文件" → "打开目录"
2. 选择文件夹

### 文件拖拽

**拖拽排序：**
- 在文件树中拖拽文件可改变顺序
- 将文件拖到文件夹上可移动到该文件夹

**拖拽上传：**
- 直接拖拽本地文件到编辑器区域

### 右键菜单

**文件右键：**
- 📄 新建文件
- 📁 新建文件夹
- 📂 打开
- ✏️ 重命名
- 📋 复制内容
- 📍 复制路径
- 💾 下载
- 🗑️ 删除

**文件夹右键：**
- 📂 展开/折叠
- 📄 新建文件
- 🗑️ 删除文件夹

## 📝 代码编辑

### 标签页管理

**基本操作：**
- 点击文件打开新标签
- 点击标签切换文件
- 点击 × 关闭标签

**拖拽排序：**
- 拖拽标签可改变顺序

**右键菜单：**
- 🚫 关闭
- 📑 关闭其他
- ➡️ 关闭右侧
- ⬅️ 关闭左侧
- 🗑️ 关闭全部

### 智能补全

Monaco Editor 提供基于语言语法的智能补全：

```javascript
// 输入 console. 会自动提示
console. // 显示 log, warn, error 等
```

**快捷键：**

**方法：**
1. 右键 → 格式化代码

## 🔍 搜索功能

### 全文搜索

**打开方式：**
1. 点击活动栏 🔍 图标
2. 快捷键 `Ctrl+Shift+F`
3. 菜单栏 → 查看 → 搜索

**搜索选项：**
- Aa 区分大小写

**搜索结果：**
- 显示匹配行号和上下文
- 点击跳转到匹配位置

### 文件搜索

在文件树顶部搜索框输入关键词：
- 支持文件名模糊匹配
- 支持文件夹名匹配

## 🐛 常见问题

### 1. Monaco Editor 加载失败

**原因：** 本地 libs 中没有 Monaco Editor

**解决：**
```bash
# 确保 libs/monaco-editor 目录存在
ls libs/monaco-editor
```

### 2. 无法打开目录

**原因：** 浏览器不支持 File System Access API

**解决：** 使用 Chrome/Edge 等现代浏览器

### 3. 保存失败

**原因：** 文件权限问题或浏览器限制

**解决：** 使用"下载文件"功能保存到本地

### 4. 智能补全不工作

**原因：** 语言未正确识别

**解决：**
1. 点击状态栏语言
2. 选择正确的语言
3. 或确保文件扩展名正确

### 5. 拖拽不生效

**原因：** 拖拽目标不支持

**解决：**
- 文件只能拖到文件夹或文件列表
- 标签只能在标签栏内拖拽
