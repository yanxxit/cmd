## 🔧 技术实现

### 测试框架
- 原生 JavaScript 实现
- 无需额外测试框架依赖
- 基于 CryptoJS 库

### 测试流程
1. 定义测试用例数组
2. 遍历执行每个测试
3. 捕获异常和错误
4. 记录测试结果
5. 更新 UI 显示

### 辅助函数
```javascript
// AES 加密
aesEncrypt(text, key, iv)

// AES 解密
aesDecrypt(encryptedText, key, iv)

// 生成随机字符串
generateRandomString(length)

// HTML 转义
escapeHtml(text)
```

## 📚 使用指南

### 运行测试
1. 打开浏览器访问测试页面
2. 点击"运行所有测试"按钮
3. 等待测试执行完成
4. 查看测试结果和详细报告

### 解读结果
- **通过**: 解密结果与原文完全一致
- **失败**: 解密结果与原文不一致或抛出异常
- **异常测试**: 预期会失败的情况（标记为⚠️）

### 重复测试
- 每次运行都会生成新的随机密钥和偏移量
- 可以多次运行验证稳定性
- 建议在不同浏览器中测试兼容性

## 🐛 已知限制

1. **浏览器兼容性**: 某些老旧浏览器可能不支持 CryptoJS
2. **性能差异**: 不同设备和浏览器性能可能有差异
3. **随机性**: 每次测试使用随机密钥，结果不可复现
