# 贷款计算器使用指南

## 📋 功能概述

贷款计算器提供两大核心功能：
1. **贷款计算** - 根据贷款金额、期限和利率计算月供和还款计划
2. **反推利率** - 根据月供反推贷款的实际年利率

## 🎯 功能一：贷款计算

### 使用场景
- 购房贷款计算
- 车贷计算
- 消费贷款计算
- 比较不同贷款方案

### 输入参数

| 参数 | 说明 | 示例值 |
|------|------|--------|
| 贷款金额 | 贷款总金额（万元） | 100 万元 |
| 贷款期限 | 贷款年限（1-30 年） | 30 年 |
| 年利率 | 贷款年利率（%） | 4.2% |
| 还款方式 | 等额本息/等额本金 | 等额本息 |

### 还款方式说明

#### 等额本息
- **特点**：每月还款金额固定
- **优势**：便于记忆和规划
- **适合人群**：收入稳定的工薪阶层
- **计算公式**：
  ```
  月供 = [贷款本金 × 月利率 × (1+ 月利率)^还款月数] ÷ [(1+ 月利率)^还款月数 - 1]
  ```

#### 等额本金
- **特点**：每月还款本金固定，利息逐月递减
- **优势**：总利息支出较少
- **适合人群**：前期还款能力强的人
- **计算公式**：
  ```
  每月月供 = (贷款本金 ÷ 还款月数) + (剩余本金 × 月利率)
  ```

### 输出结果

计算后会显示：
- ✅ **每月还款金额** - 月供数额
- ✅ **贷款总额** - 贷款本金
- ✅ **支付利息** - 总利息支出
- ✅ **还款总额** - 本金 + 利息
- ✅ **还款计划表** - 每月的详细还款明细

### 计算示例

**示例 1：商业贷款**
```

贷款期限：30 年
年利率：4.2%
还款方式：等额本息

- 每月还款：4,889.92 元
- 支付利息：76.04 万元
- 还款总额：176.04 万元
```

**示例 2：公积金贷款**
```

还款方式：等额本金

计算结果：
- 首月还款：4,133.33 元（逐月递减）
- 支付利息：49.40 万元
- 还款总额：129.40 万元
```

## 💡 使用技巧

### 1. 比较不同方案
- 使用贷款计算功能比较不同银行的贷款产品
- 对比等额本息和等额本金的总利息差异
- 考虑提前还款时，选择等额本金更划算

### 2. 识别高利贷
- 使用反推利率功能计算实际成本
- 注意"手续费"、"服务费"等隐藏费用
- 年利率超过 24% 需谨慎

### 3. 提前还款规划
- 查看还款计划表了解本金偿还进度
- 等额本息前期主要是利息，后期主要是本金
- 等额本金每月本金固定，利息逐月递减

## ⚠️ 注意事项

### 1. 利率类型
- **名义利率**：银行标称的年利率
- **实际利率**：考虑所有费用后的真实成本
- **APR**：年度百分率，包含部分费用

### 2. 还款方式选择
- **等额本息**：适合收入稳定、希望月供固定的人
- **等额本金**：适合前期还款能力强、想节省利息的人

### 3. 其他费用
计算结果不包含：
- 房屋保险费
- 贷款服务费
- 评估费
- 公证费
- 其他中介费用

### 4. 提前还款
- 部分银行会收取提前还款违约金
- 等额本息前期提前还款不划算
- 等额本金任何时候提前还款都相对划算

## 🎓 贷款知识

### 等额本息 vs 等额本金

| 对比项 | 等额本息 | 等额本金 |
|--------|----------|----------|
| 每月还款 | 固定 | 递减 |
| 总利息 | 较多 | 较少 |
| 前期压力 | 较小 | 较大 |
| 适合人群 | 工薪阶层 | 高收入人群 |

### LPR 利率
- **LPR**：贷款市场报价利率
- **定价方式**：LPR + 基点（BP）
- **调整周期**：通常每年 1 月 1 日调整

### 提前还款建议
1. **等额本息**：还款超过 1/3 期限后不建议提前还款
2. **等额本金**：还款超过 1/2 期限后不建议提前还款
3. **公积金贷款**：利率较低，不建议提前还款
4. **投资理财**：如果投资收益>贷款利率，不建议提前还款

## 📞 常见问题

### Q1: 为什么实际利率和名义利率不一样？
A: 因为可能存在手续费、服务费等额外费用，这些都会增加实际成本。

### Q2: 等额本息和等额本金哪个更划算？
A: 等额本金总利息更少，但前期压力大。选择取决于个人财务状况。

### Q3: 如何减少贷款利息？
A: 
- 选择等额本金还款方式
- 缩短贷款期限
- 提前还款（注意违约金）
- 提高首付比例

### Q4: 反推利率准确吗？
A: 反推利率基于等额本息公式计算，对于等额本金或其他还款方式可能不完全准确。
