📚 AlaSQL 语法教程

⚡ 快速开始

1. 上传数据文件（Excel、CSV 或 JSON 格式）

2. 在 SQL 编辑器中输入查询语句

3. 使用 ? 占位符代表当前数据源

4. 点击"执行查询"按钮查看结果

SELECT * FROM ? -- 查询所有数据

🔰 基础查询

查询全部 基础
SELECT * FROM ?
返回数据源中的所有记录
选择列 基础
SELECT name, age FROM ?
只返回指定的列，减少数据量
去重查询 基础
SELECT DISTINCT department FROM ?
去除重复值，返回唯一结果
限制结果 基础
SELECT * FROM ? LIMIT 10
只返回前 N 条记录

🎯 条件筛选 (WHERE)

比较条件 基础
SELECT * FROM ? WHERE age > 18
筛选年龄大于 18 的记录，支持 =, !=, <, >, <=, >=
逻辑与 基础
WHERE status = 'active' AND score >= 60
同时满足多个条件
逻辑或 基础
WHERE status = 'active' OR status = 'pending'
满足任一条件即可
模糊匹配 中级
WHERE name LIKE '%张%'
%表示任意字符，_表示单个字符
范围匹配 基础
WHERE department IN ('技术部', '销售部')
匹配列表中的任意值
区间匹配 基础
WHERE age BETWEEN 20 AND 30
匹配 20 到 30 之间的值（包含边界）

📊 排序与分页

降序排列 基础
ORDER BY salary DESC
按工资降序排列（DESC 降序，ASC 升序）
多列排序 中级
ORDER BY department ASC, salary DESC
先按部门升序，再按工资降序
分页查询 中级
LIMIT 10 OFFSET 0
每页 10 条，取第 1 页（OFFSET 从 0 开始）

📈 聚合函数

计数 基础
SELECT COUNT(*) AS total FROM ?
统计总记录数，AS 给列起别名
求和 基础
SELECT SUM(salary) AS totalSalary FROM ?
计算工资总和
平均值 基础
SELECT AVG(age) AS avgAge FROM ?
计算平均年龄
最大最小值 基础
SELECT MAX(salary), MIN(salary) FROM ?
获取最高和最低工资

🔢 分组统计 (GROUP BY)

分组计数 中级
SELECT department, COUNT(*) AS count FROM ? GROUP BY department
统计各部门人数
分组聚合 中级
SELECT department, AVG(salary) AS avgSalary FROM ? GROUP BY department
计算各部门平均工资
分组筛选 高级
GROUP BY department HAVING COUNT(*) > 5
HAVING 筛选分组后的结果，显示人数大于 5 的部门

🔄 数据处理

字符串函数 中级
UPPER(name), LOWER(name), LENGTH(name)
转大写、转小写、计算长度
计算列 基础
SELECT name, salary, salary * 12 AS annualSalary FROM ?
计算年薪（支持 + - * / 运算）
空值处理 中级
COALESCE(phone, '未登记')
如果 phone 为 NULL，返回默认值
条件表达式 高级
CASE WHEN age < 18 THEN '未成年' ELSE '成年' END
根据条件返回不同值
