## 快速验证脚本

可以使用以下 JavaScript 代码在浏览器控制台快速验证：

```javascript
// 验证生肖
function verifyZodiac(year) {
  const animals = ["鼠", "牛", "虎", "兔", "龙", "蛇", "马", "羊", "猴", "鸡", "狗", "猪"];
  return animals[(year - 4) % 12];
}

// 验证星座
function verifyConstellation(month, day) {
  const dates = [20, 19, 21, 20, 21, 22, 23, 23, 23, 24, 22, 22];
  const signs = [
    '摩羯座', '水瓶座', '双鱼座', '白羊座', '金牛座', '双子座',
    '巨蟹座', '狮子座', '处女座', '天秤座', '天蝎座', '射手座', '摩羯座'
  ];
  return dates[month - 1] < day ? signs[month] : signs[month - 1];
}

// 测试用例
console.log('2024 年生肖:', verifyZodiac(2024)); // 应该输出：龙
console.log('2000 年生肖:', verifyZodiac(2000)); // 应该输出：龙
console.log('3 月 21 日星座:', verifyConstellation(3, 21)); // 应该输出：白羊座
console.log('12 月 22 日星座:', verifyConstellation(12, 22)); // 应该输出：摩羯座
```

## 已知农历算法限制

1. **年份范围**：1900 年 - 2100 年
2. **闰月精度**：基于传统农历算法，与真实天文观测可能存在 1 天误差
3. **时区影响**：未考虑时区因素，以北京时间为准
